# 🚀 Futuristic Student CRM Dashboard
**Future Interns - Full Stack Web Development Internship (Task 2)**

A modern, responsive Student Lead Management System designed with a futuristic dark-mode aesthetic. This project allows users to manage student records, track attendance, and monitor fee statuses efficiently.

## ✨ Key Features
- **Modern UI:** Built with a "Futuristic" dark theme for a professional user experience.
- **Dynamic Student Management:** Add and view student details including ID, Course, Attendance, and Performance.
- **Data Persistence:** Uses **LocalStorage** to ensure student records are saved directly in the browser and persist after page refreshes.
- **AI-Style Status Tracking:** Includes a "Stable/Risk" status indicator for student performance monitoring.
- **Responsive Stats:** Real-time updates for "Total Records" and "Pending Fees" on the dashboard.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Icons:** Lucide-React
- **Styling:** Custom CSS/Inline Styles
- **Data Storage:** Web LocalStorage API

## 📂 Project Structure
- `/client/src/App.jsx`: Main application logic and UI.
- `/client/dist`: Production-ready build files.
- `server.js`: Node.js/Express backend setup.

## 🚀 Installation & Setup
1. Clone the repository.
2. Navigate to the client folder: `cd client`
3. Install dependencies: `npm install`
4. Run the app: `npm run dev`
5. Open `http://localhost:5173` in your browser.
